## 리눅스 명령어 정리

1. ### Top
   top 명령어는 리눅스 계열 서버의 OS 상태를 확인할 수 있는 CLI 어플리케이션이다.
   top 명령어를 통해 서버의 CPU 사용량, 메모리 사용량 등을 확인할 수 있으며, 서버에서
   구동 중인 모든 어플리케이션들을 CPU 사용률이나 메모리 사용률이 많은 순서대로 나열하여 모니터링이 가능한 명령어이다.
